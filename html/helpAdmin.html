Данная форма предназначенна для отправки заявок в фирму "Балвер" .
Для создания сообщения вам необходимо:

  1. Выбрать ошибку из списка ошибок расположенного слева.
  2. Проверить пользователя под которым будет отправляться заявка. По умолчанию ставиться пользователь системы.
  3. В случае необходимости ввести дополнительные данные в поле справа.
  4. Нажать кнопку "Отправить Заявку"
  5. После звонка тех.поддержки нажать кнопку "Ответить тех.поддержке"
